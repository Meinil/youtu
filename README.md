PHP课程设计

[演示地址](https://schrodingerseecat.github.io/youtu/#/)

使用步骤

1. 进入`front`文件夹

    ```shell
    cd front
    ```

2. 安装依赖

    ```shell
    npm install
    ```

3. 启动前端

    ```shell
    npm run dev
    ```

4. 使用`idea`导入`server`

5. 导入`resources/sql/youtu.sql`到本地的`MySQL`

6. 修改`application-dev.yml`中的数据库配置

7. 启动`main`函数，访问

    ```url
    http://127.0.0.1:3000
    ```
